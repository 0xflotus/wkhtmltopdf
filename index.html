wkhtmltopdf and wkhtmltoimage

wkhtmltopdf and wkhtmltoimage are command line tools to render HTML into PDF and various image formats using the QT Webkit rendering engine. These run entirely "headless" and do not require a display or display service.

It is yet to be fully migrated to this site, so please be patient in the interim period. Please read the auto-generated manual for wkhtmltopdf, or the API documentation for libwkhtmltox.

Current Release

The current release is 0.12.0, which was released on 6-Feb-2014. A rolling development snapshot 0.12.1-dev is also available, which includes fixes since the 0.12.0 release. The next release 0.12.1 is likely to happen after QT 4.8.6 is released and after all critical bugs in the current release have been addressed.

You can download very old releases from Google Code, but no bug reports will be accepted against them.
